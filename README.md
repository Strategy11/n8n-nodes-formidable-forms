# n8n-nodes-formidable-forms

This is an n8n community node. It lets you use _Formidable Forms Trigger_ in your n8n workflows, which receives data from Formidable Forms forms and use it in the workflow.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.
